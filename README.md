# 🎯 KPSS Aşkı

**KPSS çalışma süresi takip uygulaması — Arkadaşlarınla rekabet et, motive ol!**

> 📱 **Web uygulaması** — İster telefonda, ister bilgisayarda, her yerden çalışır!

---

## ✨ Özellikler

| Özellik | Açıklama |
|---------|----------|
| ⏱️ **Kronometre** | Start / Pause / Reset kontrollü, arka planda da sayaç işler |
| 🏆 **Leaderboard** | Haftalık ve Tüm Zamanlar sıralaması, gerçek zamanlı güncellenir |
| 🎉 **Motivasyon Rozetleri** | 1 saatten 24 saate kadar 24 baraj, her saat başı tebrik popup'ı |
| 🔥 **Anlık Rekabet** | Supabase realtime ile sıralama anında güncellenir |
| 📱 **PWA Desteği** | Ana ekrana ekle, native uygulama gibi kullan |
| 🔐 **Şifresiz Giriş** | Sadece kullanıcı adı ile kayıt ol, bir daha şifre sorma |
| 🌙 **Dark Mode** | Göz yormayan koyu tema |

## 🚀 Hızlı Başlangıç

### 1. Projeyi Klonla
```bash
git clone https://github.com/Umqra11/Kpss-A-k-.git
cd Kpss-A-k-
```

### 2. Bağımlılıkları Yükle
```bash
npm install
```

### 3. Supabase'i Kur
Supabase kurulumu için **[SUPABASE_KURULUM.md](./SUPABASE_KURULUM.md)** dosyasındaki adımları takip et.

### 4. Çalıştır
```bash
# Web için
npm run web

# Mobil için (Expo Go ile)
npm start
```
